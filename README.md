# ai-learning-notes
This repository documents my structured learning path in Artificial Intelligence, Machine Learning, Deep Learning, and the mathematical foundations behind them.

My goal is to build strong fundamentals by:

Studying core ML and DL concepts
Writing concise notes in my own words
Implementing small, minimal examples in Python/NumPy/PyTorch
Building practical intuition through code
Gradually progressing toward AI engineering and research
This repo is designed to be a long-term, evolving record of my growth as an AI engineer.

Approach
For each topic, I try to include:

A short, simplified explanation of the intuition
The key formulas
Important notes, pitfalls, and insights
A small Python/NumPy/PyTorch example (5–20 lines max)
Clear structure so that future me (or anyone reading) can understand it easily
This approach focuses on learning deeply, not just completing courses or copying slides.
